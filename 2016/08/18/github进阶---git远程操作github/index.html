github进阶---git远程对github的操控

2016.08.18 TheLuminous 初级入坑指南  热度 ℃

        github给了我们一个免费的个人空间–300M，虽然不多，但也够我们用了。我们可以用这个空间来做一些有趣的事情，比如上传自己的html代码，生成自己独特的网页（你可以用这个空间去表白我会乱说）。好啦，下面给大家讲解一下如何远程操作我们的github空间。
